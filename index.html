¿Qué es jQuery?

jQuery es una librería de JavaScript de software libre y código abierto creada en enero del 2006 para ahorrar tiempo y espacio. Usando jQuery se simplifica mucho la manera de manipular el árbol DOM, manejar eventos, interactuar con los documentos HTML, agregar animaciones...

Antes de empezar...

Antes de empezar tenemos que tener claro que no hay nada que haga jQuery que no pueda hacer JavaScript, la diferencia es que jQuery te puede llegar a ahorrar la mitad de líneas de código.

Para poder usar esta librería en nuestras páginas, tenemos dos opciones: desde un archivo local o desde un enlace externo.

Desde Local

Descargando la librería desde aquí y añadiendo la siguiente etiqueta dentro del head:

    <script type="text/javascript" src="ruta-al-archivo/jquery.js"></script>

Desde enlace externo

Añadiendo la siguiente línea en el head:

    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

Como jQuery actúa directamente sobre el árbol DOM de las páginas, es imprescindible que esta haya cargado del todo antes de que se aplique el código jQuery. Si se cargase al revés (primero jQuery y luego el DOM), no funcionaría nada. Para que jQuery se espere a que el DOM haya cargado tenemos que usar la función .ready(); de forma que todo código deberá empezar así: 

    <script type="text/javascript">
        $(document).ready(function(){
            //Código
        });
    </script>
