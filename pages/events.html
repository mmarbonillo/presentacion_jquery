Introducción Selectores DOM Eventos Efectos

Eventos

¿Qué es un evento?

Un evento es una acción realizada sobre/por la aplicación de manera impredecible y que es factible a generar una respuesta.

¿Qué eventos hay?

Eventos de ratón

    <script type="text/javascript">

        x.mouseenter()    //Evento que se dispara cuando el ratón entra en el elemento x
        x.mouseover()     //Este evento se dispara cuando el ratón entra en el elemento x o en cualquiera de sus hijos
        x.mousemove()     //Evento que se dispara cuando el ratón se mueve dentro del elemento x
        x.mousedown()     //Evento que se produce cuando se presiona un botón del ratón
        x.mouseup()       //Este evento se dispara cuando se suelta un botón pulsado del ratón
        x.click()         //Este evento solo se producirá cuando se haga mousedown y mouseup en el mismo elemento
        x.dblclick()      //Evento que se dispara al producirse dos eventos click seguidos
        x.mouseleave()    //Evento que se produce cuando el ratón abandona el elemento x
        x.mouseout()      //Este evento se dispara cuando el ratón sale del elemento x o de cualquiera de sus hijos
        x.hover(a, b)     //Esta es la forma de abreviar mouseenter y mouseout

    </script>

Eventos de teclado

    x.keydown()     //Evento que se dispara al pulsar una tecla 
    x.keypress()    //Evento que se dispara al pulsar una tecla
    x.keyup()       //Evento que se dispara al levantar una tecla pulsada

    <head>
        <script>
            $('#inputex2').on({
                keydown: function(){
                    $('#kp, #ku').css('background-color', 'unset');
                    $('#kd').css('background-color', 'blue');
                },
                keypress: function(){
                    $('#kd, #ku').css('background-color', 'unset');
                    $('#kp').css('background-color', 'blue');
                },
                keyup: function(){
                    $('#kd, #kp').css('background-color', 'unset');
                    $('#ku').css('background-color', 'blue');
                }
            });
        </script>
    </head>

Eventos de formulario

    x.focusin()     //Evento que salta cuando un elemento o algún hijo de este gana el foco
    x.focusout()    //Evento que salta cuando un elemento o algún hijo de este pierde el foco
    x.focus()       //Este evento se dispara cuando un elemento recibe el foco
    x.blur()        //Este evento se dispara cuando un elemento pierde el foco
    x.change()      //Evento que se dispara cuando cambia el valor de un elemento
    x.select()
    x.submit()
