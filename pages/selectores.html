Introducción Selectores DOM Eventos Efectos

¿Qué es un selector?

Los selectores son la forma en jQuery de seleccionar los elementos de una página.

Tipos de Selectores

Selectores Básicos

Selector de etiquetas

Con este selector accedemos a uno o varios elementos de nuestra página a través del nombre de la etiqueta.

    $('p') //Selecciona todas las etiquetas de párrafo de la página

Selector de identificador

Sirve para acceder a un elemento de la página que tenga un identificador (id) específico. Para usarlo simplemente escribimos # y el nombre del identificador. 

    $('#principal') //Estaríamos accediendo al elemento de id='principal'

Si por equivocación pusiesemos el mismo id a más de un elemento de la página, solo tendría en cuenta el primero que encuentre. Pulsa cualquiera de estos dos párrafos para verlo: 

//Ambos tienen id="parrafoEjemplo"

Selector de clase

Con este selector podemos hacer referencia a todos los elementos que tengan un nombre de clase (class) concreto.

Clica cualquier capa ;)

class='capa1'

class='capa2'

class='capa1'

class='capa2'

Concatenar selectores

Al igual que en CSS, podemos concatenar selectores usando comas para que cierta acción de ejecute en distintos elementos.

Clica cualquier capa ;)

class='clase1'

class='clase2'

class='clase3'

class='clase4'

Selector asterisco

Sirve para seleccionar todos los elementos de la página.
